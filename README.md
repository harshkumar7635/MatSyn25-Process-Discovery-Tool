# MatSyn25 Process Discovery Tool

## Overview
A Streamlit-based interactive dashboard for analyzing synthesis routes of MXenes, TMDs, and Graphene materials.

## Features
- Interactive dashboard
- Family-wise synthesis analysis
- Plotly visualizations
- Route similarity analysis
- Network analysis

## Technologies Used
- Python
- Streamlit
- Pandas
- Plotly
- NetworkX
- RapidFuzz

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
